## 数据深拷贝

> 数据深拷贝, 提取自 lodash, 最小化模块体积

Demo:

```tsx | pure
import { cloneDeep } from 'zzzjs';

const door = cloneDeep([1, 2, 3, 4, 5]);
console.log(door); // -> [1, 2, 3, 4, 5]
```
