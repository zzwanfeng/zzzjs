## getRuntimeEnv

> 获取当前运行环境,返回运行环境的集合判断

Demo:

```tsx | pure
import { getRuntimeEnv } from 'zzzjs';

const env = getRuntimeEnv();
console.log(env);
/* {
  isAndroid: false
  isChrome: true
  isEdge: false
  isIE: false
  isIE9: false
  isIOS: false
  isWeex: false
}
*/
```
